# Vention Company Leaderboard — Frontend SPA

A static single-page app that replicates Vention's internal Company Leader Board 2025 page, built with React 18 + Vite + TypeScript + TailwindCSS. All data is synthetic.

**Live demo:** https://alekseichirkov.github.io/vention-ai-challenge/

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
